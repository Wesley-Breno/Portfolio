Web crawler

Este código é uma função que simula um web crawler em Python que permite raspar as palavras de um site da web e todas as suas outras paginas, exibir a frequência dessas palavras ou procurar por uma palavra específica em todas as paginas de um site. A função usa a biblioteca requests para acessar o site e a biblioteca BeautifulSoup para analisar o HTML da página. A função também pode limitar a quantidade de páginas a serem pesquisadas e salvar o resultado da contagem das palavras como um arquivo JSON.
