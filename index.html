Analisador de seguidores do GitHub

Programa feito em Python para fazer uma análise dos seguidores do usuário. Mostrar para o usuário as pessoas que não o seguem de volta e também as pessoas que o usuário não segue de volta. Assim, é possível usar os métodos para fazer o computador deixar de seguir as pessoas que não o seguem de volta ou seguir as pessoas que o seguem, mas ele não as segue de volta.

Agenda de contatos

Desenvolvi um projeto de uma agenda de contatos utilizando o framework Django. Na agenda, é possível registrar informações como nome, sobrenome, número de telefone e descrição do contato. Além disso, foi implementada a funcionalidade de cadastro e login, permitindo que o usuário adicione seus contatos de maneira segura e possa recuperá-los caso haja perda dos dados em seu smartphone.

E-commerce

O sistema é completo e oferece uma ampla gama de funcionalidades, incluindo a capacidade de gerenciar promoções, acesso a um carrinho de compras e resumo da compra, além de gerenciar unidades de produtos disponíveis para venda. Notícias automatizadas

Site feito com o framework Django, utilizando as ferramentas requests e BeautifulSoup. Este site é responsável por fazer a raspagem de dados de sites de notícias e mostrar as notícias mais recentes destes sites, sem a necessidade de visitar cada site para ter um amplo conhecimento das notícias recentes relacionadas à tecnologia.

Web crawler

Este código é uma função que simula um web crawler em Python que permite raspar as palavras de um site da web e todas as suas outras paginas, exibir a frequência dessas palavras ou procurar por uma palavra específica em todas as paginas de um site. A função usa a biblioteca requests para acessar o site e a biblioteca BeautifulSoup para analisar o HTML da página. A função também pode limitar a quantidade de páginas a serem pesquisadas e salvar o resultado da contagem das palavras como um arquivo JSON.
